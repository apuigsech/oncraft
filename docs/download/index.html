System Requirements

  • macOS 10.15+, Windows 10+, or Linux with WebKit2GTK 4.1
  • Claude CLI installed
  • Anthropic API key configured
  • gh CLI (optional, for GitHub integration)

Installation

macOS

Open the .dmg file and drag OnCraft to your Applications folder.

Windows

Run the .msi installer and follow the prompts.

Linux

AppImage: chmod +x OnCraft_*.AppImage && ./OnCraft_*.AppImage

Debian: sudo dpkg -i OnCraft_*.deb
